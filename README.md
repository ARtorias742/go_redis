# Redis Clone in Go

A simple Redis-like in-memory key-value store implemented in Go for learning purposes.

## Features
- SET key value
- GET key
- DEL key [key ...]
- EXPIRE key seconds
- EXISTS key

1. Initialize the Go module:
   ```bash
   go mod init github.com/yourusername/redis-clone
